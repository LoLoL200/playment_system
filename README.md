# Playment System

**Playment System** is a Go-based backend application that manages user payments and transactions.  
It provides functionality for handling users, processing payments, and maintaining transaction data.

## Features
- User registration and management  
- Payment creation and tracking  
- Simple in-memory or database-backed storage  
- REST API endpoints for interaction  
- Configurable settings and logging

## Technologies Used
- **Go (Golang)**  
- **net/http** for server and routing  
- **encoding/json** for data exchange  
- **log** for logging events
